## 四、已知问题与规避

（随用户反馈与官方更新持续补充）

- **Windows**：建议使用 WSL2，纯 Windows 可能有路径与权限问题，见 [Windows 部署坑](/openclaw-knowledge/module-04-troubleshooting/01-deploy/01-windows)
- **Node 版本**：需 Node 18+，推荐 Node 22+
- **API 成本**：多任务并行可能产生较高费用，见 [Token 优化](/tutorials/help/) 与避坑指南
