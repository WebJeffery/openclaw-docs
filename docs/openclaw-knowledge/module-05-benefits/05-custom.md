# 定制化指导

> 个人 / 团队场景专属建议

针对付费用户的个性化需求，提供场景化方案建议与实施指导。

---

## 一、定制范围

| 类型 | 内容 | 交付形式 |
|------|------|----------|
| **个人场景** | 单一场景（如 A股监控、日报生成）的方案设计 | 文档 + 配置示例 |
| **多场景串联** | 跨场景工作流（如投研→报告→推送） | 架构图 + 配置清单 |
| **团队协作** | 多成员、多 Agent、权限与审批设计 | 方案文档 + 答疑 |
| **成本与选型** | 模型选型、API 成本预估、本地/云端混合 | 建议报告 |

---

## 二、预约流程

1. **提交需求**：通过专属渠道填写需求表（场景、目标、现有环境）
2. **初步评估**：1–2 个工作日内反馈是否在服务范围内、预估工作量
3. **确认排期**：双方确认时间与交付形式
4. **方案交付**：按约定提供文档、配置或连线讲解
5. **跟进答疑**：交付后一定期限内可针对方案进行有限次追问

---

## 三、交付形式

- **文档**：Markdown/PDF 方案说明、配置示例、操作步骤
- **配置模板**：可直接导入或参考的 YAML/JSON 片段
- **连线讲解**：屏幕共享，逐项讲解配置与注意事项（需预约）
- **非交付**：不包含代写代码、代部署、长期运维

---

## 四、与标准答疑的区别

| 维度 | 一对一答疑 | 定制化指导 |
|------|------------|------------|
| **问题类型** | 单点问题、故障排查 | 整体方案、场景设计 |
| **深度** | 解决当前卡点 | 覆盖选型、架构、实施路径 |
| **形式** | 文字/语音为主 | 含方案文档、可选连线 |
| **权益** | 含在基础/进阶版 | 通常需单独购买或高阶权益 |

---

## 五、适用人群

- 有明确业务目标，但不确定如何用 OpenClaw 落地
- 团队引入 OpenClaw，需要整体规划与分工建议
- 希望评估成本与 ROI，再做投入决策
- 已有初步方案，需要专家评审与优化建议
