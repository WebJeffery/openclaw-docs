# 专属资源包

> 部署脚本 / 指令合集等，一键获取

付费用户可获取经实战验证的配置与脚本，缩短从零到可用的时间。

---

## 一、资源清单

| 类型 | 内容 | 适用场景 |
|------|------|----------|
| **部署脚本** | 一键安装脚本（Linux/macOS）、Docker Compose 模板 | 本地/云部署 |
| **配置模板** | 常用通道配置、多模型切换、Cron 示例 | 快速搭建 |
| **指令合集** | 投研、办公、开发等场景的 Prompt 模板 | 提升 Agent 效果 |
| **技能包** | 精选技能配置、Gate 规则示例 | 技能定制参考 |
| **故障排查清单** | 按模块整理的检查项与命令 | 自助排障 |

---

## 二、获取方式

1. **购买后自动开通**：在知识库或专属页面查看「资源包」入口
2. **按模块解锁**：部分资源随对应模块学习进度解锁
3. **社群发放**：限时资源、更新包通过交流群推送

---

## 三、使用说明

### 部署脚本

- 执行前请阅读脚本内注释，确认环境要求（Node 版本、端口等）
- 建议先在测试环境或虚拟机中试跑
- 脚本会创建 `~/.openclaw` 等目录，请确保有写入权限

### 配置模板

- 模板中的 API Key、Token 等需替换为你的实际值
- 敏感信息勿提交到公开仓库
- 可结合 [配置参考](/tutorials/gateway/configuration-reference) 做二次调整

### 指令合集

- Prompt 需根据你的业务场景微调
- 建议先小范围测试，再推广到生产
- 可参考 [系统提示词](/tutorials/concepts/system-prompt) 优化结构

- 贡献优质资源（经审核）可获权益延长或专属致谢
