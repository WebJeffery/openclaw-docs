# 每月常规更新

> 同步版本，补新功能 / 新坑

常规更新节奏、内容预告与历史记录。

---

## 一、更新日历

- **频率**：每月约 2 次
- **时间**：通常为月中、月末，或随 OpenClaw 发版后 1–2 周内
- **形式**：知识库内更新 + 社群通知

---

## 二、常规更新内容

| 类型 | 说明 |
|------|------|
| **版本同步** | 新版本安装步骤、配置变更、Breaking Changes |
| **新功能说明** | 官方新功能的中文解读与实操指引 |
| **新坑补充** | 用户反馈的高频问题、避坑指南新增 |
| **配置示例** | 新通道、新模型、新技能的配置模板 |
| **链接修正** | 官方文档链接、命令变更等 |

---

## 三、内容预告

近期计划补充（示例，以实际为准）：

- OpenClaw 最新版本部署与迁移说明
- 新通道接入教程（若有）
- 成本优化专题补充
- 避坑指南案例扩充

---

## 四、历史记录

更新日志按时间倒序，详见 [版本日志](/openclaw-knowledge/appendix/03-changelog)。

---

## 五、如何获取更新

- **知识库内**：各模块「最近更新」标注、更新日志页
- **社群**：重要更新在交流群内通知
- **邮件/站内信**：大版本与权益相关变更（若已绑定）
