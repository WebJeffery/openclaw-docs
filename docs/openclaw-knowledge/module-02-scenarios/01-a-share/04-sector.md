# 板块主线追踪

> 规避踏空技巧

通过定时扫描 + 板块强度排序 + 资金流向分析，追踪市场主线，减少踏空风险。

---

## 一、核心思路

1. **板块强度排序**：按涨幅、成交额、资金净流入等维度排序
2. **主线识别**：连续多日领涨、资金持续流入的板块
3. **轮动预警**：主线板块出现拐点或新主线形成时提醒
4. **推送渠道**：Telegram、飞书等，便于盘中快速查看

---

## 二、Cron 配置

```json5
{
  cron: {
    timezone: "Asia/Shanghai",
    jobs: [
      {
        schedule: "0 9,15 * * 1-5",
        agent: "sector-tracker",
        message: "执行板块主线追踪：1) 获取今日板块涨跌幅、资金流向；2) 识别领涨板块与轮动信号；3) 与昨日对比，标注新晋主线；4) 输出到 session sector-daily"
      }
    ]
  }
}
```

---

## 三、板块追踪 Agent

```json5
{
  agents: {
    "sector-tracker": {
      model: { primary: "deepseek/deepseek-chat" },
      systemPrompt: {
        content: |
          你是 A股板块主线追踪助手。任务：
          1. 通过数据脚本获取板块涨跌、资金流向
          2. 按涨幅、资金净流入排序，识别当日领涨板块
          3. 与近期数据对比，标注：延续主线 / 新晋主线 / 退潮
          4. 输出格式：
             ## 今日板块主线
             ### 延续主线
             - 板块A：涨幅 X%，资金净流入 Y 亿，连续 N 日领涨
             ### 新晋主线
             - 板块B：...
             ### 退潮/观望
             - 板块C：...
      }
    }
  }
}
```

## 四、Skill 注入规则

```markdown
# ~/.openclaw/skills/sector-tracking/SKILL.md

# 板块主线追踪技能

## 指令
主线判定标准：
- 延续：连续 2 日以上涨幅居前 5，且资金净流入为正
- 新晋：当日首次进入涨幅前 5，且成交额放大
- 退潮：此前领涨板块当日跌幅居前或资金大幅流出

输出需包含板块名称、涨跌幅、资金流向、连续天数。
```

---

## 五、数据来源

- **板块数据**：通过 Tushare、iFinD 等接口获取 `ths_index`、行业指数等
- **资金流向**：部分数据源提供板块资金净流入
- **补充**：`web_search` 可搜索「今日板块涨幅」「资金流向」等公开资讯

---

## 六、规避踏空的使用建议

1. **盘中与盘后结合**：开盘 30 分钟、收盘前各扫描一次
2. **关注轮动**：主线切换时往往有先兆，新主线形成初期可重点关注
3. **结合个股**：板块领涨时，可进一步用个股分析 Agent 筛选标的
