# 用户交流群

> 经验分享 / 资源互换

付费用户专属社群，用于答疑互助、案例分享与资源互换。

---

## 一、入群方式

- 购买知识库后，在「权益中心」或订单详情页获取入群方式
- 需验证购买凭证（订单号/用户 ID）
- 入群后请按群公告修改昵称（如：昵称-模块/场景）

---

## 二、群规概要

### 鼓励

- 分享部署心得、配置技巧、踩坑经验
- 提问时附上环境与报错信息
- 回答他人问题时注明「个人经验，仅供参考」

### 禁止

- 发布广告、拉人、引流到其他付费产品
- 讨论政治、敏感话题
- 传播盗版、破解、未授权转载内容
- 人身攻击、恶意揣测

### 违规处理

首次提醒，屡犯移出群聊；严重违规可能影响后续权益。

---

## 三、活跃话题

| 话题 | 说明 |
|------|------|
| **部署与排障** | Windows/macOS/Linux 部署、通道断连、Cron 不执行等 |
| **场景落地** | A股投研、办公自动化、开发运维、自媒体工作流 |
| **技能与模型** | 技能推荐、冲突解决、Ollama 调优、多模型切换 |
| **成本优化** | Token 节省、API 中转、本地模型替代 |
| **变现交流** | 代部署、技能定制、社群运营等（不涉及具体定价拉单） |

---

## 四、社群价值

- **即时反馈**：常见问题可快速获得同侪或管理员回复
- **案例沉淀**：优质分享会整理进知识库或资源包
- **人脉连接**：同场景用户可私下交流，拓展合作
- **优先活动**：线下/线上活动名额优先开放给群成员

---

## 五、使用建议

- 善用群内搜索，避免重复提问
- 复杂问题可整理后走 [一对一答疑](/openclaw-knowledge/module-05-benefits/01-support)
- 分享时注意脱敏，勿泄露 API Key、业务数据等
